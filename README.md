1. `git checkout -b **\<branch\>**`

    This switches branches to the branch specified.  The "-b" is creating the branch.
2. `git add .`

    Adds items, or picks them up getting ready to give to git.  *NOTE: INCLUDE THE "."*
3. `git commit -m \'**\<message\>**'`

    Adds items, or picks them up getting ready to give to git.  *NOTE: INCLUDE THE "."*
